如何搭建个人博客

Created2026-01-27|Updated2026-01-29|Ravenclaw
|Post Views:

写在前面

很早之前就想搭建一个自己的博客网站，但是之前由于转专业忙忙碌碌，暂时放下这个计划。终于到了寒假可以自由地学习！用了小半个下午的时间完成了这个博客的初步建设。速速来总结搭建的过程以及遇到的一些问题！

个人博客的好处

  • 把自己所学的知识进行总结
  • 记录美好生活
  • 见证自己的成长
  • 帮助其他人避开自己踩过的坑

基础知识

搭建过程中所需要的知识并不多，基本跟着视频或者文档一步一步走都可以实现，视频底下的评论区也非常之有用。但是，在后面对博客网站进行更多的个性化设计的时候，需要对前端开发有一些基本的了解（我现在也不会），但是无所谓慢慢来!

下面是一些在这个阶段要了解的：

  • Markdown语法
  • json格式

    参考视频：带你快速了解Json以及Json的用法-哔哩哔哩

  • GitHub
  • Vscode

    可以安装多种插件，对不同类型的文件进行编辑，并且还有终端命令行，对于管理博客数据非常方便

搭建个人博客的参考视频：

  • 零基础搭建个人博客！手把手教你用Hexo+GitHub+Zeabur打造属于你的网站!-哔哩哔哩
  • 手把手教你搭建属于自己的hexo+github博客-哔哩哔哩

框架

Hexo+GitHub+Vscode

搭建流程

安装Node.js

Node.js是一个免费、开源、跨平台的 JavaScript 运行时环境，让开发者可以创建服务器、Web 应用、命令行工具和脚本。

  • 进入Node.js的官方网站https://node.org.cn/en进入下载页面，下载对应的长期支持版本(LST)的msi文件
  • 下载完成后双击msi文件，大部分按默认配置即可
  • 检查是否安装成功(出现下面结果即成功)

注册GitHub

进入GitHub官方网站https://github.com/完成注册即可

  • 创建一个叫做username.github.io的Repository

    若username不为自己的账户名，会导致输入该网址显示404。

Git for win安装

  • 进入Git for win的官方网站https://gitforwindows.org/直接下载即可
  • 打开cmd进行验证正确安装 之后所有操作都可以在Vscode的Git bash终端中进行即可

GitHub SHH配置

  • 在Git bash中输入
  • 进入C盘-用户-username-.ssh文件夹，用记事本方式打开id_sra.pub文件，复制全部内容
  • 双击GitHub个人主页头像，点开setting，找到SSH and GPG keys，将复制内容输入即可
